# @hilum/blocks

CLI for adding Hilum marketing blocks to an app.

## Install

```bash
pnpm add -D @hilum/blocks
```

## Usage

```bash
pnpm hilum list
pnpm hilum add hero-simple-centered
```

`hilum add` writes the selected block into `src/components/blocks` by default and installs any missing dependencies declared by the block registry.
